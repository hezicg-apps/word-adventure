# 🎓 WordMaster Express - ללמוד אנגלית בכיף

מערכת אינטראקטיבית ללימוד אוצר מילים באנגלית המשלבת כרטיסיות חכמות, מבחני ידע ומשחקים אסטרטגיים. הפרויקט נבנה במטרה להפוך את שינון המילים לחוויה מהנה ומניעה לפעולה.

## ✨ תכונות עיקריות

* **הזנה פשוטה:** הדבקה מהירה של רשימת מילים בפורמט `מילה - תרגום`.
* **למידה הדרגתית:**
    * **כרטיסיות (Flashcards):** שינון מילים עם אפשרות להקראה קולית (TTS).
    * **מבחן ידע:** חידון אמריקאי לקבלת ציון ושחרור נעילת המשחקים.
* **מרכז משחקים (Game Center):**
    * **משחק זיכרון:** התאמת זוגות של מילה ותרגום.
    * **4 בשורה (Connect 4):** משחק אסטרטגיה נגד המחשב או נגד חבר (באותו מכשיר), שבו כל מהלך דורש מענה על שאלה.
    * **הקוד הסודי (WordQuest):** משחק ניחוש מילים בסגנון Wordle המבוסס על אוצר המילים שהוזן.
* **מצב לילה/יום:** התאמה אישית של ממשק המשתמש.

## 🚀 איך מפעילים?

1.  מעלים את הקבצים לשרת או פותחים את ה-GitHub Pages של הפרויקט.
2.  במסך הראשי, לוחצים על **"בואו נתחיל"**.
3.  מדביקים את רשימת המילים (לדוגמה: `Apple - תפוח`).
4.  עוברים על הכרטיסיות ומסמנים "יודע/ת".
5.  מבצעים את המבחן. ברגע שמגיעים ל-**70% הצלחה**, המשחקים נפתחים!

## 🛠 טכנולוגיות

* **HTML5 & CSS3** (עם Tailwind CSS לעיצוב מהיר ומודרני).
* **Vanilla JavaScript** - ניהול מצב (State) ללא ספריות חיצוניות.
* **Speech Synthesis API** - להקראה קולית של המילים.
* **Canvas Confetti** - לחגיגות ניצחון.
